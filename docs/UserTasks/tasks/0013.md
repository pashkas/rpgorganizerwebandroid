# 0013 — Simplify-ревью: imgLoaded баг, пропущенный сброс чеклиста, чистка импортов

## Запрос пользователя
«/simplify» — ревью всех изменённых файлов на переиспользование, качество и эффективность.

## Цель
Найти и исправить проблемы качества в текущих изменениях (чеклисты, masonry→grid, свайпы, красная рамка и др.).

## Найденные проблемы и подход

### 1. Баг: `imgLoaded` терялся при пересоздании массива `skillsGlobal$`
Свойство `skill.imgLoaded` выставлялось динамически на объекте `GlobalItem` через `(load)` в шаблоне. Но `skillsGlobal$` пересоздаёт массив при каждом обновлении — свойство терялось. С OnPush мутация объекта также не тригерила CD.

**Решение:** заменили динамическое свойство на `Set<string> loadedSkillImgs` в компоненте `MainWindowComponent`, привязанный к `skill.tskId`. Set живёт в компоненте и не теряется при пересоздании массива. Событие `(load)` тригерит CD корректно.

### 2. Баг: `onTskDateChange` не сбрасывал `checklistItems.isDone`
При смене даты задачи сбрасывались только `isDone` у подзадач, но чеклист-айтемы оставались отмеченными. В `qwickSetDate` сброс был, а в `onTskDateChange` — нет.

**Решение:** добавлен сброс `checklistItems.forEach(ci => ci.isDone = false)` в `onTskDateChange`, аналогично `qwickSetDate`. Делегировать в `srv.setStatesNotDone` нельзя — тот также сбрасывает `isActive`, что не нужно в этом контексте.

### 3. Неиспользуемый импорт `CdkDragDrop`
В `add-item-dialog.component.ts` импортировался `CdkDragDrop`, но использовался только `moveItemInArray`.

**Решение:** убран из импорта.

### 4. Пустые строки внутри однострочных guard-ов
В `add-item-dialog` и `customSwipe.directive` были лишние пустые строки перед `return` внутри коротких `if`-блоков.

**Решение:** убраны.

## Затронутые файлы
- `src/app/main-window/main-window.component.ts` — добавлен `loadedSkillImgs: Set<string>`
- `src/app/main-window/main-window.component.html` — `(load)` пишет в Set, условие рамки читает из Set
- `src/app/task-detail/task-detail.component.ts` — добавлен сброс чеклиста в `onTskDateChange`
- `src/app/add-item-dialog/add-item-dialog.component.ts` — убран `CdkDragDrop`, пустая строка в guard-е
- `src/app/customSwipe.directive.ts` — убраны пустые строки в guard-ах
