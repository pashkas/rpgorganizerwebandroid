# level-up-msg: ОП слева, ОН справа — убрать дублирование

## Запрос пользователя
«В окне levelUp которое из changeService запускается ОП и ОН друг под другом с двух сторон. Сделай чтобы слева было ОП (star), а справа ОН (алмаз)»

## Цель
В `level-up-msg` по обе стороны от главной картинки раньше стояли одинаковые столбики `star + perkPoints` и `diamond + abPoints`. Пользователь хотел разнести валюты по сторонам — слева только ОП, справа только ОН.

## Подход
В `level-up-msg.component.html` левый столбец оставил только с `star + perkPoints` (ngIf по `perkPoints != null`), правый — только с `diamond + abPoints` (ngIf по `abPoints != null`). В правом блоке порядок внутри ряда перевернул: сначала иконка алмаза, потом число (чтобы число не прилипало к центральной картинке). Класс `.secImg` и жёлтый цвет цифры ОП сохранены.

## Затронуто
- Файлы: `src/app/level-up-msg/level-up-msg.component.html`

## Побочные эффекты
- Если `perkPoints == null` (ОП отключены в `GameSettings.isAbPointsEnabled`), левый столбик просто не рисуется — раньше он рисовался внутри внешнего `*ngIf="abPoints != null"`, теперь условия независимые.
- Наблюдение 0012 описывало макет `level-up-msg` как «вертикальный стэк star + diamond» с двух сторон — после этой правки стэк расщеплён. Дописал обновление в 0012.

## Корректировки от пользователя
Не было, закрыл реакцией «+».
