# Прокачка навыка из главного окна перса не должна открывать страницу задачи

## Запрос пользователя
«Если поднимать навык из главного окна перса (upAbility) он прокачивается и открывается окно задачи (навыка). Пусть не открывается, только прокачивается и показывает изменения.»

## Подход
В `pers-list.component.ts` метод `upAbilityFromList` после `upAbil(ab)` дополнительно вызывал `showAbility(ab)`, который через `PersService.showAbility` навигировал на `/pers/task/:id`. Убран вызов `showAbility` — остался только апгрейд (попап изменений `PerschangesService` отрабатывает сам).

## Затронуто
- Файлы: `src/app/pers-list/pers-list.component.ts`
- Методы: `PersListComponent.upAbilityFromList`
