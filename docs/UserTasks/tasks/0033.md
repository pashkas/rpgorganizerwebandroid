# Чеклист под «Дни недели» в настройке стейтов навыка

## Запрос пользователя

> «В настройке навыков сделай в настройке стейтов чеклист под "дни недели".»

## Подход

В [add-item-dialog.component.html](src/app/add-item-dialog/add-item-dialog.component.html) поменял местами два блока: теперь сверху «Дни недели» (`data.showWeekDays`), ниже «Чеклист» (`data.showChecklist`). Этот же диалог используется и для обычных задач, и для подзадач-стейтов — флаги `showChecklist`/`showWeekDays` выставляет вызывающий код, так что порядок меняется для всех сценариев одновременно, что ок.

## Затронуто

- Файлы: `src/app/add-item-dialog/add-item-dialog.component.html`
