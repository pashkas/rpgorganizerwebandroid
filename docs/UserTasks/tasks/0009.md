# 0009 — Выравнивание padding контейнера masonry с gap между элементами

## Запрос пользователя
«В главном окне в масонри немного не красиво. Надо чтобы отступ по бокам и сверху снизу у контейнера был такой же как и между элементами»

## Проблема
CSS Grid `.masonry-grid` имел `gap: 2px` между элементами, но обёрточный `.masonry-container` не имел padding — элементы прижимались вплотную к краям контейнера, выглядело несимметрично.

## Решение
Добавлен `padding: 2px` к `.masonry-container` в `src/styles.css`, чтобы внешние отступы совпадали с внутренними (`gap`).

## Затронутые файлы
- `src/styles.css` — класс `.masonry-container`
