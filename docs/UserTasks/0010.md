# 0010 — Красная рамка появляется только после загрузки картинки

## Запрос пользователя
«Не нравится, что красная рамка подгружается раньше, чем загружаются картинки»

## Цель
Убрать визуальный артефакт — красная рамка (failCounter > 0) в masonry навыков отрисовывалась до загрузки картинки, что выглядело некрасиво.

## Подход
Добавили событие `(load)` на `<img>`, которое выставляет динамическое свойство `skill.imgLoaded = true`. Условие отрисовки рамки дополнено проверкой этого флага: рамка появляется только когда `failCounter > 0 && skill.imgLoaded`.

Свойство `imgLoaded` не добавляется в модель — оно динамически создаётся на объекте `skill` при срабатывании события load.

## Затронутые файлы
- `src/app/main-window/main-window.component.html` — добавлен `(load)="skill.imgLoaded = true"`, условие рамки дополнено `&& skill.imgLoaded`
