# 0001 — Выравнивание чеклистов с подзадачами в task-detail

## Запрос пользователя
«Чеклисты находятся правее чем сама подзадача (курсивом). Сделай чтоб на одном уровне.»

## Проблема
В компоненте `task-detail` элементы чеклиста (`checklistItems`) отображались с отступом `ml-4` (margin-left Bootstrap), из-за чего они визуально сдвигались правее относительно названия подзадачи.

## Решение
Убран класс `ml-4` у контейнеров чеклистов в обоих режимах:
- **Режим просмотра** (строка ~121): `<div class="ml-4" *ngFor="let ci of st.checklistItems">` → убран `ml-4`
- **Режим редактирования** (строка ~320): `<div *ngIf="st.isChecklist ..." class="ml-4">` → убран `ml-4`
