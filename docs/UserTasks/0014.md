# 0014 — Simplify-ревью: OnPush + Set баг, мёртвый код, форматирование

## Запрос пользователя
Запуск `/simplify` для ревью всех изменённых файлов.

## Цель
Выявить и исправить проблемы с переиспользованием кода, качеством и эффективностью в текущем диффе.

## Что нашли и исправили

### 1. Баг: OnPush + мутация Set (main-window.component)
`loadedSkillImgs.add(skill.tskId)` вызывался прямо в шаблоне через `(load)`. При `ChangeDetectionStrategy.OnPush` Angular не видит мутацию `Set` (ссылка не меняется), поэтому красная рамка `[ngStyle]` не появлялась после загрузки картинки.

**Решение:** вынесли в метод `onSkillImgLoad(tskId)`, который создаёт новый `Set` и вызывает `cdr.markForCheck()`.

### 2. Мёртвый код: `let time = null` (task-detail.component)
Переменная объявлена, но нигде не используется в `addStateToTask()`. Удалена.

### 3. Мёртвый тернарник (task-detail.component)
`header += tsk.requrense == "нет" ? " подзадачу" : " подзадачу"` — обе ветки возвращали одно и то же. Упрощено до `header += " подзадачу"`.

### 4. Двойные пустые строки
Убраны лишние пустые строки в `customSwipe.directive.ts` и `main-window.component.ts`.

## Что проверили, но НЕ меняли
- **Дублирование сброса чеклистов** в `task-detail` (onTskDateChange/qwickSetDate) vs `setStatesNotDone` в сервисе — сервисный метод дополнительно сбрасывает `isActive`, что меняет семантику. Требует решения пользователя.
- **Stringly-typed `'дни недели'`** — системная проблема по всему проекту, не введена этим диффом.
- **`ok()` возвращает string vs object** — намеренная обратная совместимость для разных вызывающих.
- **`GlobalItem.isNoActive/failCounter`** — стандартные проекции view-model.

## Затронутые файлы
- `src/app/main-window/main-window.component.ts` — добавлен `ChangeDetectorRef`, метод `onSkillImgLoad`
- `src/app/main-window/main-window.component.html` — `(load)` вызывает метод вместо inline-мутации
- `src/app/task-detail/task-detail.component.ts` — удалены `let time` и тавтологический тернарник
- `src/app/customSwipe.directive.ts` — убрана лишняя пустая строка
