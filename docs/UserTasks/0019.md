# 0019 — Изменение формулы ограничения прокачки навыка по уровню персонажа

## Запрос пользователя
«Сделай чтобы можно было прокачивать до 2, за каждые целые 10 уровней ещё 2»

## Проблема
Старая формула: `max = floor(persLevel / 10) + 3` — давала начальный потолок 3 и прибавляла по 1 за каждые 10 уровней перса.

## Решение
Новая формула: `max = 2 + floor(persLevel / 10) * 2`

| Уровень перса | Макс. уровень навыка |
|---------------|---------------------|
| 0–9           | 2                   |
| 10–19         | 4                   |
| 20–29         | 6                   |
| 30–39         | 8                   |
| 40–49         | 10                  |

## Затронутые файлы
- `src/app/pers.service.ts` — метод `recountAbilMayUp`, строка с вычислением `max`

## Примечания
- Ограничение действует только на обычные навыки; перки ограничены только `maxAbilLvl`
- Механика `mayUp` не изменена — просто другой порог блокировки
