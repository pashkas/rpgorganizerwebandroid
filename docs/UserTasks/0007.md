# 0007 — Доработка красной рамки для пропущенных задач в masonry навыков

## Запрос пользователя
«Рамка не красивая — должна быть без закругления, чуть толще и ровно по границам картинки»

## Цель
Улучшить визуал красной рамки (failCounter > 0) в masonry навыков: убрать закругление, сделать толще, и чтобы рамка шла точно по краю картинки без зазоров.

## Подход
- Рамку перенесли с внешнего `div ngxMasonryItem` на сам элемент `<img>` — так она идёт ровно по границам картинки, без зазоров от padding кнопки и flex-контейнера.
- Убрали `border-radius` (был 4px).
- Толщину подобрали итеративно: 1px → 4px → 2px → 3px → 4px → **3px** (финал).

## Корректировки от пользователя
- Первый вариант (4px на div) — рамка не прилегала к картинке, был зазор. Перенесли на img.
- Подбирали толщину: 2px → 3px → 4px → остановились на 3px.

## Затронутые файлы
- `src/app/main-window/main-window.component.html` — `[ngStyle]` с рамкой перенесён с div на img, параметры: `3px solid red`, без border-radius
