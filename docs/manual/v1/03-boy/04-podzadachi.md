# 3.4 Подзадачи, таймеры и счётчики

> *«Большую задачу невозможно съесть за один укус. Её можно только разрезать и съесть по кусочкам. В РПГ Органайзере — ровно по этому принципу.»*

---

Есть задачи, которые **нельзя решить одним «тапом на меч»**. Например, «Утренний ритуал» — это не одно действие, а цепочка: умыться, зарядка, душ, завтрак. Каждый пункт — отдельное действие, и пропустить один не то же самое, что пропустить все.

Для этого есть **подзадачи** (они же `states` в коде) и вспомогательные механики.

---

## Подзадачи

Подзадача — это **маленькая задача внутри задачи**. Настраивается в форме задачи, раздел «Подзадачи».

### Как выглядит

В карточке задачи появляется список чекбоксов:

```
☐ Умыться
☐ Зарядка 5 минут
☐ Душ
☐ Полезный завтрак
```

Тыкаешь в пункт → галочка → пункт закрашен.

### Когда задача считается выполненной

Зависит от настроек (чипы над списком подзадач):

- **Название** — в заголовок задачи подставляется имя *выполненной* подзадачи. Удобно, когда хочешь видеть «что именно сейчас делаешь».
- **Заголовок** — склеивает все подзадачи в заголовок.
- **Сумма (∑)** — значения подзадач суммируются. Полезно, когда подзадачи — это штуки «отжимания 10 / отжимания 15 / отжимания 20», и тебе важен общий счёт.
- **Рефреш (⟳)** — счётчик подзадач сбрасывается в начале дня. Так каждое утро ты начинаешь с чистыми чекбоксами.

---

## Подзадача с днями недели

У каждой подзадачи можно указать **конкретные дни недели** (`tskWeekDays`). Например:

- «Бегать» — только пн / ср / пт.
- «Плавать» — только вт / чт.

В другие дни эти подзадачи **не показываются**. Удобно, когда у тебя сложное расписание: одна большая задача «Кардио», а внутри распределение по дням.

---

## Чеклист внутри подзадачи

Уровень «подзадача» — это один клик. Но есть ещё **чеклист** (`isChecklist` + `checklistItems`) — это уже список пунктов с галочками **внутри** подзадачи.

Подходит, когда подзадача включает несколько конкретных шагов:

```
Подзадача: «Уборка кухни»
  ☐ Посуда
  ☐ Плита
  ☐ Холодильник
  ☐ Пол
```

Тыкаешь в каждый пункт — галочка. Пункт становится вычеркнутым. Когда все пункты закрыты — подзадача считается выполненной.

---

## Таймер ⌛

Кнопка с песочными часами. Нажал → пошёл отсчёт. Делаешь своё дело, не отвлекаешься. Когда дошёл до **aimTimer** (цели по времени):

- Таймер **остановится сам**.
- Если включён **будильник** (`isAlarmEnable`) — запищит или завибрирует.
- Задача сразу засчитается как выполненная.

Таймер **паузится**, если свернуть приложение (в зависимости от версии). Для надёжности держи приложение открытым.

**Где использовать:**
- Медитация (10 мин)
- Чтение (20 мин)
- Глубокая работа (50 мин Pomodoro)
- Учёба (30 мин)

### Хитрости таймера

- Прервать досрочно → тапнуть ещё раз. Засчитается частичное время.
- Если сделал больше цели → таймер продолжит идти, и цель просто будет «перевыполнена».

---

## Счётчик ✓

Кнопка с галочкой. Нажал → +1 к счётчику. Нажал ещё раз → ещё +1. Пока не достигнешь `aimCounter` (цель).

**Где использовать:**
- Отжимания (30 раз — тапаешь 30 раз, хотя обычно набиваешь подходами и тыкаешь 3 раза на пачку по 10).
- Стаканы воды (8 стаканов в день — тыкаешь каждый раз, когда выпил).
- Иностранные слова (15 в день).
- Подходы по чему-то.

### Минусом

Случайно нажал лишний раз? В настройках задачи обычно есть **кнопка «−»** для уменьшения счётчика. Или долгий тап по «+» — сбросить.

---

## Комбинация: таймер + счётчик + подзадачи

Да, можно **всё сразу**. Задача «Глобальная утренняя рутина» может иметь:
- Подзадачи на компоненты рутины.
- Таймер на каждую (5 мин зарядка, 10 мин медитация).
- Общий счётчик «дней подряд без пропусков».

Но не злоупотребляй — слишком сложная задача превращается в приложение внутри приложения. Ставь себе **минимальную структуру**, которая нужна.

---

## Когда подзадачи реально нужны

- **Когда задача разбивается на разнородные шаги.** «Утренняя рутина» — разные действия, каждое отдельно.
- **Когда хочешь видеть прогресс внутри задачи.** «Недельная уборка: кухня / ванная / гостиная / спальня» — галочки показывают «сколько осталось».
- **Когда нужно разное расписание по дням.** «Кардио» = «Бег пн/ср/пт» + «Плавание вт/чт».

**Когда НЕ нужны:**
- Задача простая и одноразовая (счётчика хватит).
- Подзадач больше 5–6 — лучше разбить на разные навыки.

---

## Сумма и «общий счёт»

Чип **∑ (сумма)** в настройках подзадач — это когда ты хочешь, чтобы **значения счётчиков подзадач складывались в родительский счётчик**.

Пример: задача «Отжимания 30», внутри подзадачи:
- Подход утром (10)
- Подход днём (10)
- Подход вечером (10)

Включаешь сумму → в каждой подзадаче свой счётчик, а сверху видишь «всего сделано 30/30». Удобно.

---

## Рефреш ⟳

Чип «рефреш» — **счётчик подзадач обнуляется каждое утро**. Это то, что превращает разовое выполнение в ежедневное:

- Без рефреша: отметил галочки — остались до конца времён.
- С рефрешем: каждое утро чистый лист, галочки надо ставить заново.

**Большинство повседневных задач** должны иметь рефреш. Без него они станут «одноразовыми».
