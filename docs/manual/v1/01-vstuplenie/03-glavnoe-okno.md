# 1.3 Главное окно: твоя арена

> *«В любой игре 99% времени ты проводишь в одной локации. В нашей — вот эта.»*

---

После входа ты окажешься в **Главном окне**. Пока оно пустое — герой ещё не создан, задач нет, монстров нет. Давай разберём, что где находится, чтобы, когда всё появится, ты уже знал, куда смотреть.

## Что ты видишь

```
┌─────────────────────────────────────────┐
│         [имя героя]   [уровень]         │  ← шапка
├─────────────────────────────────────────┤
│                                         │
│                                         │
│      ┌──────┐  ┌──────┐  ┌──────┐       │
│      │ нвк1 │  │ нвк2 │  │ нвк3 │       │  ← сетка задач/навыков
│      └──────┘  └──────┘  └──────┘       │
│      ┌──────┐  ┌──────┐  ┌──────┐       │
│      │ нвк4 │  │ нвк5 │  │ нвк6 │       │
│      └──────┘  └──────┘  └──────┘       │
│                                         │
├─────────────────────────────────────────┤
│ [👤]  [↩]  [⚙]  [⚔]  [🛡]  [🔀]  [✚]  │  ← панель управления
└─────────────────────────────────────────┘
```

### Верх — шапка героя

Имя, аватарка, текущий уровень, XP-бар, золото, монстры рядом (если встретились). Красиво, информативно, постоянно напоминает, кем ты стал.

### Середина — поле боя

Здесь живут твои **задачи** или **квесты** — в зависимости от выбранного режима. По умолчанию это **masonry-сетка плиток**: каждая плитка это один навык или квест, картинка, прогресс-бар, какие-то мелкие значки состояния.

Двойной тап по сетке — **быстро переключиться между "навыки" и "квесты"**. Не знал? Теперь знаешь.

### Низ — панель управления

Набор кнопок, каждая со своим смыслом:

| Кнопка | Что делает |
|---|---|
| 👤 **Аватарка героя** | Перейти на экран персонажа (там твой профиль, характеристики, настройки). |
| ↩ **Первый / глобальный** | Вернуть фокус на первую задачу или на общий вид. |
| ➡ **Следующий** | Перейти к следующей задаче. |
| 🔀 **Навыки ↔ Квесты** | Переключить, что показывается: навыки или квесты. |
| ⚙ **Шестерёнка / редактировать** | Войти в режим сортировки: задачи можно двигать драгом. |
| ⚔ **Меч (Сделано)** | Отметить текущую задачу выполненной. Получаешь опыт и золото. |
| 🛡 **Щит (Провал)** | Отметить текущую задачу проваленной. Теряешь опыт и прогресс. |
| ✚ **Плюсик** | Быстрое добавление задачи/навыка/квеста. |

---

## Свайпы: геймпад в твоём пальце

Главное окно поддерживает свайпы по картинке текущей задачи. Это быстрее, чем тыкать в кнопки:

- **Свайп вверх ⬆** — *Сделано* (эквивалент кнопки «Меч»).
- **Свайп вниз ⬇** — *Провалено* (эквивалент кнопки «Щит»).
- **Свайп горизонтальный ↔** — переключение между «первая задача» и «глобальный вид сетки».

Свайпы настроены на минимум 50 пикселей и максимум 800 мс. Случайное касание ничего не делает — так что можно спокойно тыкать в экран, не боясь «нечаянно запороть задачу».

---

## Долгий тап

Ещё одна полезная штука: **долгое нажатие на аватарку героя** мгновенно открывает окно «Добавить задачу». Идеально, когда мысль пришла, а в меню лезть лень.

---

## А что там наверху справа?

Обычно там живут контекстные кнопки: смена вида (карты), кнопка синхронизации (☁), переход к дневнику параметров (если включён). В разных режимах чуть разная подборка — проще один раз обежать все меню глазами, чтобы запомнить.

---

## Но пока тут пусто

Сетка пустая. Героя нет. Ну и ладно — давай уже, наконец, его создадим.
