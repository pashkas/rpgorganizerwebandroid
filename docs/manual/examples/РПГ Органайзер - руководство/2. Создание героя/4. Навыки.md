# Навыки

Мудрецы прошлого говорят, что основные качества в мире РПГ можно развить только совершенствуя свои привычки. Чтож, добавим к каждой характеристике несколько дел, которые ты будешь выполнять на регулярной основе и назовем их "навыками". 

Каждый навык "привязан" к определенной характеристике. Если ты хочешь прокачать характеристику - развивай навыки из которых она состоит. Например - навык "утренняя зарядка" будет качать характеристику "Телосложение", а навыки "Чтение" и "Творчество" будут прокачивать "Интелект". Получается, что характеристика - это результат совокупности навыков.

Еще раз повторюсь - навыки развиваются путем многократного повторения. Воспринимай навык как привычку. В мире РПГ Органайзера НАВЫК = ПРИВЫЧКА.

Вот тебе еще несколько примеров:
* **Иностранный язык.** Чтобы развить этот навык, тебе будет нужно приобрести ПРИВЫЧКУ читать на иностранном языке по нескольку минут в день, постепенно увеличивая длительность.
* **Отжимания.** Чтобы развить этот навык, тебе будет нужно приобрести ПРИВЫЧКУ каждый день делать по нескольку отжиманий, постепенно увеличивая их количество.
* **Питание.** Чтобы развить этот навык, тебе будет нужно каждый день правильно питаться, постепенно усложняя свои требования.
